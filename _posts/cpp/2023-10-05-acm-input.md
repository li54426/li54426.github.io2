### 2 进阶

#### 2.1 输入方式为每行一个数组

```
1,2,3
4,5,6
```



```c++
//空格分割
12 3       
45
0 0 0 0 0

#include<sstream>
#include<string>
#include<iostream>

int main{
    string line;
    while(getline(cin, line)) {  //依靠geline读入一行
       stringstream ss(line);    //每行先读入字符串，然后再区分每个数字
        int num;
        int sum=0;  
        while(ss >> num)
           sum += num;   //计算总和
    }
}

```
